# Researcher-Information-System


A system that displays the relevant data about researcher and their publications. 
Built in OOP style using Model-View-Controller, with Database and the Model, View as the View, and Control and Research(class definitions) as the Controller. 
Built using C# and XAML in .NET environment.


Requires the latest MySQL Connector .NET drivers to work. 
When run in visual Studio, will need to add the reference of Mysql.data to the project.
